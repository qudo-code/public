
# Sleep Work
![](https://raw.githubusercontent.com/qudo-code/public/refs/heads/main/media/Pasted%20image%2020251203125553.png)
When you think of a fix for that bug while trying to fall asleep.  
  
The characters in each row are generated roughly based on a reference image, but there is a high chance of randomness. This means that each time you view this NFT it will be a unique experience.  
  
Interaction  
- Click the stats bar to reveal settings.  
- Change the duration of each frame with the "Speed" option.  
- Pause on a certain frame using the "Pause/Play" button. When paused, the NFT will continue to rerender the selected frame based on the set speed.  
- Change color themes with the "Theme" option.
